# The ACCurate Syncer

A clean starter for a browser extension focused on syncing live ACC commentary with sports streams.

## Status

This repository is intentionally stripped back. Sync behavior is not implemented yet.

## Current scaffold

- Manifest V3 extension shell
- Minimal popup UI
- Basic lint and test scripts

## Run locally

1. Install dependencies with `npm install`
2. Open `chrome://extensions/` and enable Developer mode
3. Click Load unpacked and select `extension/`
4. Open the popup to confirm the scaffold loads
